# Design Pattern Guidelines

1. **Mỗi người chọn ít nhất 2 Design Pattern.**  
2. Giải thích lý do chọn từng pattern.  
3. Minh họa cách áp dụng pattern đó trong dự án (đoạn code mẫu, hoặc ví dụ).  
4. Đặt tên file theo quy tắc: `DesignPattern_<TênThànhViên>.md`  
   Ví dụ: `DesignPattern_Vuong.md`  
5. Vote chọn 5 design patterns
